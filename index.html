JavaScript daqui a 20 anos

É difícil imaginar como vai ser a linguagem de web daqui a 20 anos, mas vivenciamos no mundo tecnológico. Mas essa ideia de prever o futuro, ajuda na questão de idealizar alguns projetos, para os próximos anos. Concordamos que é difícil prever um cenário futurista, mas eu aposto em um sistema operacional feito com Java Script. Aposto que cada vez mais o JavaScript vai controlar as coisas de seu device ou computador. Java é uma das linguagens mais promissoras em termos de comunidade, novas funcionalidades e até invenções de empresas. Daqui a pouco a programação vai fazer parte do ensino fundamental nas escolas, o JavaScript tem mais chance de ser uma linguagem adotada por ser mais flexível.
